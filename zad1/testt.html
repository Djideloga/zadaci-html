Web čitači

Web čitač je aplikacija koja je namenjena prikazu sadržaja web stranica. Stranice se mogu dobaviti kako iz fajl-sistema,tako i putem TCP/IP mreže. Protokol kojim komuniciraju web čitači i web server je HTTP protokol. Danas su najpoznatiji sledeći web čitači:

  1. Microsoft Internet Explorer
  2. Mozila
  3. Opera
